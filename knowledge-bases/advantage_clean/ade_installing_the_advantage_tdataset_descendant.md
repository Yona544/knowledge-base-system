The Advantage TDataSet Descendant is a "BDE replacement" that allows you to create and distribute a database application that does not require the BDE.

- [Installed Files for Advantage TDataSet Descendant](ade_installed_files_for_advantage_tdataset_descendant.md)lists the files installed and their default location.

One of the following file groups will also be installed depending on the product selected:

- [Delphi Specific Files Installed for the Advantage TDataSet Descendant](ade_delphi_specific_files_installed_for_the_advantage_tdataset_descendant.md)

- [C++Builder Specific Files Installed for the Advantage TDataSet Descendant](ade_c_builder_specific_files_installed_for_the_advantage_tdataset_descendant.md)

See [Uninstalling the Advantage TDataSet Descendant](ade_uninstalling_the_advantage_tdataset_descendant.md) for information about what files are uninstalled by executing the uninstall program and how to fully undo this installation from your system.
