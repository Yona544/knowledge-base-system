# Error 5077 Ae Cyclic Relation

5077 AE\_CYCLIC\_RELATION

5077 AE\_CYCLIC\_RELATION

Advantage Error Guide

| 5077 AE\_CYCLIC\_RELATION  Advantage Error Guide |  |  |  |  |

A relation cannot result in a cycle with a child descending from itself.
