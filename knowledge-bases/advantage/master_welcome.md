Advantage Help files are shipped separately with each product, but you should always have the following minimum set that must be located together in the same directory:

For Windows Help:

ADVANTAGE.hlp

ADVANTAGE.cnt

ADSERROR.hlp

ADSERROR.cnt

For Web Help:

ADVANTAGE.htm

ADVANTAGE

ADSERROR.htm

ADSERROR
