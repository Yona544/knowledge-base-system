| 7078 The Advantage Database Server cannot authenticate the user  Advantage Error Guide |  |  |  |  |

Problem: This error was caused by one of the following.

|  |  |
| --- | --- |
| 路 | The user name and password supplied when connecting to an Advantage Data Dictionary were incorrect. |

|  |  |
| --- | --- |
| 路 | The user name and password were not supplied but the Data Dictionary has been set up to require user authentication. |

Solution:

|  |  |
| --- | --- |
| 路 | Verify the user name and password is correct. |

|  |  |
| --- | --- |
| 路 | Set up the Advantage Data Dictionary so that it does not require user authentication. |
