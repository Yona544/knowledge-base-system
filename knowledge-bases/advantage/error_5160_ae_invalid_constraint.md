| 5160 AE\_INVALID\_CONSTRAINT  Advantage Error Guide |  |  |  |  |

One of the following conditions is true:

|  |  |
| --- | --- |
| 路 | The field-level or record-level constraint is invalid. It is either of the wrong data type or invalid syntax. |

|  |  |
| --- | --- |
| 路 | Cannot set a default field value to NULL when a field constraint exists that does not allow NULL values to exist in the column. |

|  |  |
| --- | --- |
| 路 | The default field value must be set to a non-NULL value before setting the field constraint that does not allow NULL values to exist in the column. |

|  |  |
| --- | --- |
| 路 | An invalid date, time, or timestamp value or syntax was specified for the field constraint. |
